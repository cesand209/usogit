Practicas con git y github

1) pra crear un repositorio con git usamos el comando git init

2) Una vez realizamos modificaciones en los archivos debemos registrar los cambios 2 luego hacer el commit para que se haga efectivo

  • 2-a) para registrar una modificacion usa el comando git add .
  • 2-b) parahacer el commmit ell comando git commit -m "descripcion del commit"

3) para ver todos los commit usar git reflog

4) para ver las ramas del repositorio git branch

5) para posiicionarce en una rama especifica git checkout < nombrerama >

6) para saber el estado de los archivos por ejemplo i fueron modificados pero no se registro el cambio en git usar git status -s
